GitHub, Inc. adalah penyedia hosting Internet untuk pengembangan perangkat lunak dan kontrol versi menggunakan Git .
Ia menawarkan kontrol versi terdistribusi dan fungsi manajemen kode sumber (SCM) dari Git, ditambah fitur-fiturnya sendiri.
Ini menyediakan kontrol akses dan beberapa fitur kolaborasi seperti pelacakan bug , permintaan fitur , manajemen tugas , integrasi berkelanjutan dan wiki untuk setiap proyek.
 Berkantor pusat di California , telah menjadi anak perusahaanMicrosoft sejak 2018.
 
 GitHub menawarkan layanan dasarnya secara gratis. Layanan profesional dan perusahaannya yang lebih maju bersifat komersial. [5] Akun GitHub gratis biasanya digunakan untuk menghosting proyek sumber terbuka . [6] Mulai Januari 2019, GitHub menawarkan repositori pribadi tak terbatas untuk semua paket, termasuk akun gratis, tetapi hanya mengizinkan hingga tiga kolaborator per repositori secara gratis. [7] Mulai dari 15 April 2020, paket gratis mengizinkan kolaborator tak terbatas, tetapi membatasi repositori pribadi hingga 2.000 menit Tindakan GitHub [8] per bulan. [9] Pada Januari 2020, GitHub melaporkan memiliki lebih dari 40 juta pengguna [10] dan lebih dari 190 jutarepositori [11] (termasuk setidaknya 28 juta repositori publik), [12] menjadikannya host kode sumber terbesar di dunia. 

Layanan GitHub dikembangkan oleh Chris Wanstrath , PJ Hyett , Tom Preston-Werner dan Scott Chacon menggunakan Ruby on Rails , dan dimulai pada Februari 2008. Perusahaan, GitHub, Inc., telah ada sejak 2007 dan berlokasi di San Francisco.
